Always use single quotes for strings in all project files (TypeScript, TSX, etc.).

**Why:** User preference for this project's code style.

**How to apply:** When writing any new code or editing existing code, use single quotes (`'`) instead of double quotes (`"`). This applies to imports, JSX props, string literals, etc. Biome is the formatter — single quotes should be set in the config already.
